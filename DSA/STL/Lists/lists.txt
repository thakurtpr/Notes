<---------Basics--------->

A list in C++ STL (Standard Template Library) is a dynamic data structure that allows 
efficient insertion and deletion of elements at any position.
It is a doubly-linked list, meaning each element points to both the previous and next element.

<---------Basics--------->



<---------Common Operations--------->

Insert: Inserts a new element at a specified position in the list.
Erase: Removes an element at a specified position in the list.
Push_back: Adds a new element to the end of the list.
Push_front: Adds a new element to the beginning of the list.
Pop_back: Removes the last element from the list.
Pop_front: Removes the first element from the list.
Sort: Sorts the elements in the list.
Reverse: Reverses the order of elements in the list.
